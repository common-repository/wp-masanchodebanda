== Description ==

Add a ribbon on the top left corner of your blog and support the "Más Ancho de Banda" Campaign

== Installation ==
1. Upload contents of the ZIP file to your wp-content/plugins/ folder.
2. Activate the "WP MasAnchoDeBanda" plugin.

== Notes ==

The http://masanchodebanda.com campaign is an information campaing that looks to alert people and authorities in Uruguay about the high prices and low quality internet services. Please refer to http://masanchodebanda.com to read more.
